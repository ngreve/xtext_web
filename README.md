# How to run this project
The web directory contains a front- and backend to run the Xtext project in the de.nicogreve.dsl.expressions.parent directory.

## Frontend
```
cd web/frontend
npm install
npm run serve
```

## Backend
```
cd web/backend
npm install
npm run start
```
